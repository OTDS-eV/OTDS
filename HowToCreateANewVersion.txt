1. Aktuelle OTDS Version über Mercurial downloaden
2. Versionsnummernkreis erweitern bzw. anpassen
3. Inkrementelle XSDs erstellen (
	- Ausführen C:\OpenDataStructure\XML-Tools\Automatische_Inkrementelle_XSD_Konvertierung\Make_Increments.bat
	- Dateien kopieren aus C:\OpenDataStructure\XML-Tools\Automatische_Inkrementelle_XSD_Konvertierung\Result
	- Einfügen unter C:\OpenDataStructure\Sourcen\xsd\incremental_xsds

4. XSDs mit integrierte Examples validieren
	- Altova öffnen
	- C:\OpenDataStructure\XML-Tools\Validierung_aller_Examples.xsl
	- Ausführen des XSL mit F10
	- Datei "C:\OpenDataStructure\XML-Tools\Alle_example_initial_delivery.xml" auswählen
	- Datei "C:\OpenDataStructure\XML-Tools\Alle_example_update.xml" auswählen
	- Datei "C:\OpenDataStructure\XML-Tools\AlleExamples.xml" auswählen

5. Validieren ggf. alle vorliegende Daten (FTI, DEM, 5VF, ALD, SLR)

6. Packen der Dateien in eine Datei otds_Vx_x.zip
	Zip mit Unterordner XSD und example

7. Gepackte Version intern ablegen


8. Gepackte Version an das Technische Forum weiterleiten
